# Алгоритм решения

## Метод GMRES

GMRES (Generalized Minimal RESidual) - итерационный метод для решения 
систем линейных уравнений Ax = b с несимметричными матрицами.

### Основные шаги:

1. **Инициализация:**
   - r₀ = b - Ax₀
   - β = ||r₀||₂
   - v₁ = r₀ / β

2. **Построение ортонормированного базиса:**
   - Для j = 1, 2, ..., m:
     - w = Avⱼ
     - Ортогонализация Грама-Шмидта

3. **Решение задачи наименьших квадратов:**
   - min||βe₁ - Hₘy||₂

4. **Обновление решения:**
   - xₘ = x₀ + Vₘyₘ

### Преимущества:
- Сходимость для несимметричных матриц
- Монотонное уменьшение невязки
- Устойчивость к округлениям

### Предобуславливание:
Используются различные предобуславливатели:
- Jacobi
- ILU
- Без предобуславливания
